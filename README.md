# SMART-GARAGE-SYSTEM

## Â Smart garage is one such device that helps ease the access of the garage doors in domestic use. Smart garage system controls its objects such as windows, doors, lights, and also few gas detectors, alarm using micro-controller or computer technology. Smart garage help you to access your doors with the help of a computable device such as smart phones or tablets connected to a network, and access by means of secured pins to authorize the access. In this project we will be implementing Smart garage system with IOT and we will be showing the implementation of Smart garage system using Cisco Packet Tracer.
